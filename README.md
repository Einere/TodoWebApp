## About TodoWebApp
TodoWebApp is todo SPA programmed by Vue.js, Material Design Bootstrap 4.    


## Requirements
- npm

## Build

1. clone or download.
```shell
git clone https://github.com/Einere/TodoWebApp.git
```

2. go to project directory.
```shell
cd TodoWebApp
```

3. type "npm install"
```shell
npm install
```
4. type "npm run dev"
```shell
npm run dev
```
