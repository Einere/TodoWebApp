## About TodoWebApp
TodoWebApp is todo SPA programmed by Vue.js, Material Design Bootstrap 4.    
This app use Webpack server instead of Node.js.  


## How to build
### Requirements
- npm

### Build
1. clone or download.
2. go to project directory.
3. type "npm run dev"
